## 🌿 About the Event

The **IRF World Ranger Congress** is the premier global gathering for rangers and conservation professionals. The 2027 edition will be held in **Puerto Iguazú, Argentina**, from **April 19-23, 2027**, bringing together guardians of nature from around the world beneath the majestic Iguazú Falls.

This landing page serves as the digital gateway to this important conservation event, showcasing the mission, venue, and significance of the congress.

## 🚀 Getting Started

### Prerequisites

- **Node.js** (>=18.0.0)
- **pnpm** (>=8.0.0)

### Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/your-username/irf-world-ranger-congress-2027.git
   cd irf-world-ranger-congress-2027
   ```

2. **Install dependencies**
   ```bash
   pnpm install
   ```

3. **Start development server**
   ```bash
   pnpm dev
   ```

4. **Open your browser**
   Navigate to `http://localhost:8080`

## 📜 Available Scripts
